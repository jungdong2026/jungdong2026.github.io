FUZZY THEORY 

퍼지이론(FUZZY THEORY)
퍼지이론:
    퍼지집합이론은 1965년, 미국 버클리대학의 교수인 Lofti A. Zadeh에 의하여 제안된 이론으로
    원소에 소속도의 개념을 도입하여 애매한 자료들까지도 하나의 집합으로 구성할 수 있다는 이론이다.
    여기서는 퍼지집합의 정의, 표현방법 등에 대하여 소개한다. 퍼지 집합에도 크리스프집합과 마찬가지로 연산을 정의할 수 있다.
    소속도를 이용하여 퍼지 집합 사이에 관계를 정의할 수 있다. 여기서는 관계의 표현, 반사관계, 대칭관계, 추이관계가 정의된다.

퍼지집합:
    전체집합 U의 임의의 부분집합을 A라 하면, 
    A의 특성함수를 fA(x)라 할 때, x가 A의 원소이면, fA(x)의 값은 1이고,
    x가 A의 원소가 아니면 0이다.
    
    이를테면, 전체집합 U = {a, b, c, d, e, f, g, h, k}에 속하는 한 부분집합 A = {c, d, e, f}라 할 때,
    특성함수의 값을 써서 집합 A를 나타내면 A = {0, 0, 1, 1, 1, 1, 0, 0, 0} 와 같다.
    
    집합의 원소의 포함관계를 나타내는 특성함수의 개념을 일반화하여 임의의 집합 A에 속하는 모든 원소 x마다
    구간 [0, 1]에 속하는 한 실수를 대응시키고 속하지 않는 원소에는 0을 대응시키는 경우를 생각하자.
    이 때 이런 집합을 퍼지집합(fuzzy set)이라 하고 그 원소를 소속원(member),
    각 소속원에 대응시킨 실수값을 그 소속원의 소속도(membership grade)라 부른다.
    
    퍼지집합은 영문자 A, B, C, . . .로, 그의 소속원은 소문자 a, b, c, . . .로 나타낸다.
    각 소속원 x의 소속도는 함수를 μA(x)써서 나타내고 이것을 소속함수(membership function)라 한다.
    퍼지집합을 소속원과 소속도를 써서 표현할 때는 다음과 같은 표현을 사용한다.

            A = μA(a)/a + μA(b)/b + … + μA(z)/z
        
    [예 1-1] 다음은 퍼지집합을 나타낸다. 이 때 소속도가 0인 소속원은 나타내지 않는다.

            A = 0.1/a + 0.4/b + 0.7/c + 0.9/d + 1/e
            {젊은이} = 0.3/10세 + 0.5/15세 + 1/20세 + 0.9/30세 + 0.7/40세 + 0.2/50세

    퍼지집합을 그림으로 나타낼 때는 x축에 소속원을, y축에 소속도를 써서 나타낸다. 
    퍼지집합에서 소속원이 어떤 연속인 구간을 가지고 있을 때, 그 소속도는 연속인 소속함수로 나타낼 수 있다. 
    따라서 그 그래프는 연속인 그래프가 된다.

    [예1-2] 퍼지집합 A = {x | x는 실수 5에 가까운 실수}이고 
    소속함수가 A = μA(a)/a + μA(b)/b + … + μA(z)/z로 주어져 있을 때,

    이것을 그래프로 나타내면,

            A = μA(a)/a + μA(b)/b + … + μA(z)/z
            μA(5) = 1
            μA(4) = 0.5
            μA(3) = 0.2
            μA(-∞) = 0가 된다.

퍼지집합에서의 특수한 용어:
    위에서와 같이 퍼지집합을 정의할 때,
    퍼지집합에 관한 여러 가지 용어를 다음과 같이 정의한다.
    특히 퍼지집합을 논의할 때, 전통적인 집합을 크리스프집합(crisp set)이라 하여 구별한다.
    즉, 크리스프집합이란 소속도가 없는 단순한 원소들의 집합 또는 소속도가 모두 1인 집합이다.

1. α - cut
    퍼지집합 A에서 소속도가 보다 크거나 같은 모든 소속원의 크리스프집합을 이라 하고 기호 로 나타낸다. 즉,

    Aα = {X|μA(x)} ≥ 0, x ∈ A}

2. 지지대
    퍼지집합 A에서 소속도의 값이 0보다 큰 소속원의 크리스프집합을 집합 A의 지지대(support)라 하고 
    Supp A로 나타낸다. 즉,

    Supp A = {X|μA(x)} > 0, x ∈ A}

3. 퍼지집합의 크기
    크리스프집합에서는 원소의 개수를 그 집합의 크기라고 한다. 
    그러나 퍼지집합에서는 소속원의 개수가 의미가 없으므로 소속도의 총합으로 크기를 정의하고 |A|로 나타낸다. 즉,

    |A| = Σx∈AμA(x)

     [예 1-3] 퍼지집합 A = 0/a + 0.1/b + 1/c + 0.2/d + 0.6/e + 0.8/f + 0/g에 대하여 다음을 구하여라.

        (1)A0.2, A0.2  (2) Supp A   (3) |A|
 
    <해>
        (1) 소속도가 0.2보다 크거나 같은 소속원은 c, d, e, f이므로
            A0.2 = {c, d, e, f} 마찬가지로 A0.6 = {c, e, f}

        (2) 소속도가 0보다 큰 소속원은 b, c, d, e, f 이므로
            Supp A = {b, c, d, e, f}

        (3) 퍼지집합 A의 크기는 소속도의 총합이므로
            Supp A = 0.1 + 1 + 0.2 + 0.6 + 0.8 = 2.7

퍼지수:
    퍼지이론에서는 수(number)도 하나의 퍼지집합으로 여길 때가 있다.
    이를테면, [예 1-2]에서와 같이 5근방의 수를 퍼지집합으로 나타낼 수 있다.
    이와 같이 하나의 수를 기점으로 하여 퍼지집합을 나타낼 그 수를 퍼지수(fuzzy number)라 한다.
    퍼지수 a의 소속도는 직선 x = a에 대칭인 연속함수를 써서 나타내거나, x = a에 대칭인 이등변 삼각형으로 나타낸다.
