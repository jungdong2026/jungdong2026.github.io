샤프닝은 블러링의 반대 효과를 가집니다. 샤프닝은 영상강화라고도 하는데, 영상에서 상세한 부분들을 더욱 강조하는 효과를 갖게 됩니다. 영상을 샤프닝 하는 것은 대비 효과를 증가시키게 됩니다.

회선 마스크는 중심에서는 양의 값을 가지며 바깥 경계는 음의 값을 갖게 됩니다. 일반적은 샤프닝 마스크로는 다음과 같은 것이 있습니다.

0 -1  0
-1 5 -1
0 -1  0

-1 -1 -1
-1  9 -1
-1 -1 -1

1 -2  1
-2 5 -2
1 -2  1

영상을 샤프닝하는 것은 고주파 통과 필터에 기반을 합니다. 고주파 통과 필터는 저주파 요소들을 제거하고 아주 세세한 부분들만을 보여주는 효과를 가집니다. 일반적인 고주파 통과 필터는 다음과 같습니다.

{-1/9, -1/9, -1/9, -1/9, 8/9, -1/9, -1/9, -1/9, -1/9}

여기서 계수의 합이 0 이 됨을 주목하세요...
