2차 미분의 장점은 밝기값이 점차적으로 변화되는 영역에 대해서는 반응을 보이지 않는다는 것입니다. 점차적으로 밝기가 변화되는 영역은 에지라고 볼 수 없기때문입니다.

2차 미분 연산자 중에 라플라시안 연산자가 있는데 이 연산자는 잡음에 상당히 민감합니다.

* 라플라시안 연산자

{0, -1, 0, -1, 4, -1, 0, -1, 0} 또는 {-1, -1, -1, -1, 8, -1, -1, -1, -1}

잡음에 덜 민감한 2차 미분 에지 검출기는 라플라시안-가우시안(Laplacian of Gaussian)연산자입니다. 여기서는 7X7, 9X9 LoG Operator를 이용하여 에지를 검출해 보겠습니다.

 

7X7 LoG Operator

옆의 메뉴중 회선에있는 소소코드를 수정하세요.

int kheight=7,kwidth=7;
int bias=0;
float kernel[49]={ 0,0,-1,-1,-1,0,0,0,-2,-3,-3,-3,-2,0,-1,-3,5,5,5,-3,-1,-1,-3,5,16,5,-3,-1,
-1,-3,5,5,5,-3,-1,0,-2,-3,-3,-3,-2,0,0,0,-1,-1,-1,0,0};
